# File Creation Feature - Quick Usage Guide

## How to Create Files

### Step 1: Create a Folder (if you haven't already)
1. Look at the left navigation sidebar
2. Find the "Folders" section
3. Click the "+" button to create a new folder
4. Enter a folder name and click "Create folder"

### Step 2: Access the Create Menu
1. Find your folder in the left sidebar
2. Click the **3 dots (‚ãØ)** icon next to the folder name
3. In the dropdown menu, hover over **"Create in folder"**
4. A submenu will appear with file type options

### Step 3: Choose a File Type
Select from the following options:

#### üìã **List**

#### üìù **Markdown (.md)**

### Step 4: Name Your File
1. A prompt will appear asking for the file name
2. Enter a descriptive name
3. Click OK or press Enter

### Step 5: Edit Your File
The editor will open automatically (for editable file types):

#### Text Editor Features
- Type directly in the editor
- Changes auto-save after 3 seconds
- Click "Export" to download as .txt

#### Markdown Editor Features
- Toggle between Edit, Split, and Preview modes
- Write markdown syntax (##, **, -, etc.)
- See live preview on the right (in Split mode)
- Click "Export" to download as .md

#### DOCX Editor Features
- Use the toolbar for formatting:
  - **H1, H2, H3** - Headings
  - **B** - Bold (or Ctrl+B)
  - **I** - Italic (or Ctrl+I)
  - **‚Ä¢** - Bullet list
  - **1.** - Numbered list
  - **üîó** - Add link
- Click "Export" to download as .docx

#### Spreadsheet Editor Features
- Click any cell to edit
- Use "Add Row" to add a new row
- Use "Add Column" to add a new column
- Click "Export" to download as .xlsx

### Step 6: Save and Close
1. Your changes auto-save every 3 seconds
2. Click the **X** button to close the editor
3. If you have unsaved changes, you'll get a confirmation dialog

## Viewing and Opening Files

### View Files in a Folder
1. Click the folder name or the arrow (‚ñ∂) to expand it
2. Files will appear indented under the folder
3. Each file shows its icon and name

### Open a File
1. Click on any file in the folder
2. The appropriate editor will open
3. Edit and save as needed

## Tips and Tricks

### Auto-Save
- All editors auto-save after 3 seconds of no typing
- Look for "(Unsaved changes)" indicator in the header
- When it disappears, your changes are saved

### Keyboard Shortcuts
- **Ctrl+B** - Bold text (DOCX editor)
- **Ctrl+I** - Italic text (DOCX editor)
- **Ctrl+S** - Manual save (browser default, may trigger download)

### Export Files
- Click the "Export" button in any editor
- File downloads to your default download folder
- Filename matches what you entered

### Dark Mode
- All editors support dark mode
- Toggle dark mode in the sidebar
- Editor theme updates automatically

### Unsaved Changes Warning
- If you try to close with unsaved changes
- You'll see a confirmation dialog
- Choose to stay and save, or close without saving

4. Export as .docx when ready to share

## Troubleshooting

### File Not Saving
- Wait at least 3 seconds after typing
- Check for "(Unsaved changes)" indicator
- Try clicking Export to force a save

### Can't Find My File
- Make sure the folder is expanded (click the arrow)
- Check you're in the correct workspace
- Files are stored per workspace

### Editor Not Opening
- Try refreshing the page
- Check browser console for errors
- Ensure JavaScript is enabled

### Export Not Working
- Check your browser's download settings
- Allow downloads from this site
- Check your Downloads folder

## Need Help?

If you encounter issues:
1. Check the browser console (F12) for errors
2. Try refreshing the page
3. Clear browser cache and localStorage
4. Contact support with specific error messages
