## ğŸ” Troubleshooting

### "I don't see the Folders section"
**Solution:** The sidebar might be collapsed. Look for a collapse/expand button at the top of the sidebar.

### "I don't see the + New button"
**Solution:** Make sure you're looking in the **Folders** section, not in other sections. The button is small and appears to the right of the word "Folders".

### "The button doesn't work"
**Solution:** 
1. Make sure you have a workspace selected
2. Refresh the page
3. Check browser console for errors

### "I created a file but can't find it"
**Solution:** 
1. If you created it in a folder, expand that folder
2. If you created it at root level, it should appear in the Folders section
3. Files are stored in localStorage - check your browser storage
