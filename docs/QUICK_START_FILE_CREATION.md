## â“ Troubleshooting

### File not saving?
- Wait 3+ seconds after typing
- Check for "(Unsaved changes)"
- Try Export to force save

### Can't find file?
- Expand the folder (click arrow)
- Check correct workspace
- Files are per-workspace

### Editor won't open?
- Refresh the page (F5)
- Check browser console (F12)
- Ensure JavaScript enabled

### Export not working?
- Check download settings
- Allow downloads from site
- Check Downloads folder
